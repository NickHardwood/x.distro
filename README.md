### Project Structure

Downloading the source distribution from Github creates the following file structure;
````
{root}
  +-- base
  +-- config
  +-- doc
  +-- include
  +-- scripts
  +-- spike
  +-- src
  |    +-- main.c
  +-- test
  +-- <.editorconfig>>
  +-- <.gitattributes>
  +-- <.gitignore>
  +-- <makefile>
  +-- <readme.md>
````

Downloading the release distribution from Github creates the following file structure;
````
{root}
  +-- rootfs.tar.gz
  +-- rootfs
  +-- <ubuntu.exe>
  +-- <license.txt>
  +-- <versions.txt>
````
